#  Be Focussed - PyQT Desktop app for Windows and Linux

## Packages :
 - imutils
 - keras 
 - tensorflow
 - mkl-service
 - pyqt5
 - sqlite3

## How to use:
 - Install all the required packages as mentioned above.
 - Open terminal or command prompt in "app" folder.
 - Run command "python3 main.py".
